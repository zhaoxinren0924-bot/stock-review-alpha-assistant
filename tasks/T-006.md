## 目标(What)

一句话: 实现股票详情页的标题栏、概览区域和复盘记录表格。

具体:
- **标题栏**：股票名称 · 基本面看板、股票代码 · 行业 · 日期、"+ 新增假设"按钮、"季度复盘"按钮。
- **概览卡片行（4列）**：
  - **假设状态概览**：显示各状态数量分布（稳定 X / 待观察 X / 风险 X），不显示抽象分数。
  - **当前结论**：文本展示投资结论。
  - **下次复盘**：日期 + 复盘类型标签。
  - **持仓信息**：仓位%、浮盈亏%、股息率、PB（从 Position API 获取，前端计算）。
- **复盘记录表格**：日期 / 主题 / 结论 / 下一步 / 类型，从 Review API 获取。
  - 类型区分：复盘（蓝色标签）、检查项（灰色标签）。
  - "查看全部记录"链接。

---

## 不在范围内

明确**不**做的事:
- 不实现假设健康度 72/100 的抽象分数（用状态概览替代）。

- 不实现复盘记录的排序/筛选。
- 不实现"查看全部记录"的跳转（仅视觉）。
