## 目标(What)

一句话: 安装前端路由和 shadcn/ui 组件库，建立组件目录结构，初始化设计系统。

具体:
- 安装 `react-router-dom` 并配置基础路由。
- 初始化 `shadcn/ui`（New York 风格），安装设计图所需的全部基础组件。
- 建立清晰的组件/页面/hooks/services 目录结构。
- 初始化设计系统常量（颜色、间距、圆角等 CSS 变量或 Tailwind 配置）。

## 注意事项

- `shadcn/ui` 初始化时会修改 `tailwind.config.js` 和 `tsconfig.json` 的 path alias，确认 `vite.config.ts` 中的 `@` 解析保持一致。
- 单命令启动是不变量：`python main.py` 后前端开发服务器仍需手动 `npm run dev`，但构建产物不能破坏后端启动流程。
- 组件安装选择 New York 风格（设计图偏现代紧凑）。
- **圆角统一 8px**：所有组件使用 `rounded-lg`，禁止 `rounded-xl`（`docs/design-system.md` §4.1）。
